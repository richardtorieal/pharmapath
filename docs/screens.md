# PharmaPath — Screens

## HomeScreen (`src/screens/HomeScreen.jsx`)

### Purpose
Drug library: browse, filter, and search all drugs to select one for detailed exploration.

### Layout
```
Hero banner ("From Molecule to Market")
Category filter bar  [All | Cardiovascular | Antibiotic | Neuro | Metabolic]
Search input
Drug card grid  (auto-fill, minmax 280px)
PGx & Racial Equity educational callout (bottom)
```

### Drug Card
Each card displays:
- Drug name (generic) + brand name
- Indication (short)
- Drug class
- Formulation type
- FDA approval year
- Badges: `🧬 PGx`, `★ Race-Specific`, `3D Viewer`, chemical formula

### Filtering Logic
```javascript
DRUGS.filter(d =>
  (filter === "all" || d.cat === filter) &&
  (!search || (d.name + d.brand + d.cls + d.ind).toLowerCase().includes(search.toLowerCase()))
)
```

### Navigation
- Click any drug card → `onSelect(drug)` → App routes to DrugScreen
- "Open Drug Designer" button → `onDesigner()` → App routes to DesignerScreen

### Bottom Callout
Static educational panel about:
- Underrepresentation of minorities in clinical trials
- Ancestral allele frequency differences (CYP2D6, G6PD, VKORC1)
- Importance of equitable prescribing

## DrugScreen (`src/screens/DrugScreen.jsx`)

### Purpose
Deep-dive into a single drug: 3D molecular viewer, drug info panel, and a 6-step lifecycle journey.

### Layout
```
Header: drug name + brand + class + year + badges
Back arrow (→ HomeScreen)

Two-column section:
  Left column:   MolViewer (Three.js 3D viewer, 400px height)
                 Chemical formula label (top-left overlay)
  Right column:  Drug info panel (indication, class, mfr, year, formula, mol. weight)
                 Element legend (LegendChip per element in molecule)
                 PGxSection (if drug.pgx === true)

AtomPanel (slides in below columns when an atom is selected)

Lifecycle Stepper:
  Step nav buttons  [← Ideation | Pharmo-Chem | Preclinical | Clinical | Regulatory | Post-Market →]
  Progress counter  "Step N of 6"
  DepthToggle       [L2 Overview | L3 Technical]
  Content area      (max-height 420px, scrollable)
    → MD component rendering LC_CONTENT[step][depth](drug)
```

### Lifecycle Steps

| Index | Key | Title | Icon |
|-------|-----|-------|------|
| 0 | ideation | Target Ideation & Discovery | 🔬 |
| 1 | pharmchem | Pharmaceutical Chemistry | ⚗️ |
| 2 | preclinical | Preclinical Testing | 🐭 |
| 3 | clinical | Clinical Trials | 🏥 |
| 4 | regulatory | Regulatory Review | 📋 |
| 5 | postmarket | Post-Market Surveillance | 📊 |

### Depth Levels
- **L2 (Overview):** Plain-language explanation suitable for general audiences
- **L3 (Technical):** Detailed pharmacological/chemical content for professionals/students

Content is generated by calling `LC_CONTENT[step.key].L2(drug)` or `.L3(drug)` — functions that return markdown strings. Some content branches by `drug.id` or `drug.cat`.

### PGx Regulatory Banner
When `drug.pgx === true` and the user is on the **Regulatory** step, a special warning banner is rendered: "⚠ This drug has pharmacogenomic labeling requirements."

### State
```javascript
step     // 0–5, current lifecycle step
depth    // "L2" | "L3"
selAtom  // null | { element, role, desc, color } — drives AtomPanel visibility
```
